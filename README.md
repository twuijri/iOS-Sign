# iOS-Sign





Usage
------

You can use the tool to sign programs and upload them on your site with the manufacture of the file and all the (plist) needed


# Must be modified

* `DEVELOPER="iPhone Distribution: XXXX XXXXX (XXXXXXXXXXX)"`
* `BUNDLEID="com.myWebSite.$APPNAME"`
* `ipServer="root@192.0.0.0"`
* `userNameServer="root"`
* `linkFoldar="/home/public_html"`
* `WebSite="myWebSite.com"`
